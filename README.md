# Python Dual Calculator

A beginner-friendly Python calculator that features both a Command-Line Interface (CLI) and a Graphical User Interface (GUI). 

This project is to practice core programming logic, functions, and version control with Git.

## Features

* **Two versions available:** * `calculator.py`: A text-based calculator running directly in the terminal.
  * `gui_calculator.py`: A fully interactive graphical version.
* **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
* **Advanced Operations:** Square roots (`√`), percentages (`%`), and powers (`^`).
* **Error Handling:** Built-in safeguards against dividing by zero and calculating the square root of negative numbers.

## Built With

* **Python 3.x:** The core programming language.
* **Tkinter:** Standard Python interface to the Tk GUI toolkit (used for the graphical version).
* **Math Module:** Standard Python library for advanced mathematical operations.
